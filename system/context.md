# Project Context

## Project Name
Hiragana Quiz

## Project Type
A lightweight web application for learning Japanese Hiragana.

## Goal
Help users memorize Hiragana characters through timed multiple-choice quizzes.

## Target Users
Beginner Japanese learners.

## Core Learning Mechanic
Users see a Hiragana character and must select the correct romaji reading from multiple choices.

## Platform
Browser-based application (HTML, CSS, JavaScript).

## Constraints
- No frameworks (pure HTML/CSS/JS).
- Must run locally without server.
- Code must remain simple and readable.
- All quiz logic must be separated from UI rendering.
